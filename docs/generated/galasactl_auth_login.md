## galasactl auth login

Authenticate against a Galasa ecosystem

### Synopsis

Log in to a Galasa ecosystem using an existing access token

```
galasactl auth login [flags]
```

### Options

```
  -b, --bootstrap string   Bootstrap URL
  -h, --help               help for login
```
